# 🗣️ SHANE - Voice Digital Assistant (Improved Version)

A modernized, user-friendly **voice-controlled digital assistant** built in Python, running in JupyterLab.

### ✨ Features
- Real microphone voice input (no wake word required)
- Significantly louder text-to-speech using gTTS + pydub
- Open popular websites (YouTube, Twitter/X, Reddit, Instagram, Gmail, WhatsApp, etc.)
- Music support: Apple Music, Amazon Music, and song search on YouTube Music
- Weather updates (Mumbai)
- Current time and date
- Smart Google search for any query
- Clean interactive chat + voice UI

### 🎯 Example Commands
- `open twitter` / `open reddit` / `open apple music`
- `play song perfect` 
- `what time is it`
- `what is the weather`
- `how are you`
